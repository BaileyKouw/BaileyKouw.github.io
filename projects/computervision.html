Project

Video Feed Color Detection

This Python project takes user input to choose a color. Next, it changes the video feed to HSV and isolates the color from the user's input. We used Python and OpenCV to create this application. This was later useful in several robotics projects that used this application to find a color and then avoid that isolated color on the floor when navigating an obstacle course.
