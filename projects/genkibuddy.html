GENKI BUDDY

Tools

VSCode   •   React Native / Expo   •   Adobe XD, Illustrator, and Photoshop

Design

I started by coming up with a list of the top features I wanted my app to have. I settled on three main sections: Vocabulary, Verbs, and Dialogue. From there I started laying out my app and its design in Adobe XD. Once I had everything designed and layed out I began researching React Native and working on some test projects.

Development

After I felt more confident in React Native, I started developing my project. I started by creating the basic home screens, and used React Native stack navigation to make my layout switch between screens. During my first phase I kept the dialogue and vocab words in json files to be read in by the app. The final thing I got to was creating the flash cards to flip and show the word and translation.

Future

This app is still a work in progress and I intend to continue developing it/ I hope to get the time to implement a database for the vocabulary words/dialogues. I also intend to make the vocabulary section more responsive to words that the user gets incorrect by showing them a “missed words” or “words you know” section within each chapter.
